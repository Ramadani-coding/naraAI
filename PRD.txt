# PRD — NARA

## Windows AI Desktop Assistant dengan Voice, HUD, dan OpenAI Codex

---

# 1. Ringkasan Produk

## 1.1 Nama Produk

**NARA**

Kemungkinan kepanjangan nama:

```text
NARA = Neural Agentic Runtime Assistant
```

Atau versi yang lebih simpel:

```text
NARA = Natural AI Runtime Assistant
```

Untuk sekarang kita pakai:

> **NARA — Neural Agentic Runtime Assistant**

---

## 1.2 Deskripsi Singkat

**NARA** adalah aplikasi AI assistant untuk **Windows 10** yang berjalan di komputer pengguna, memiliki **HUD futuristik**, bisa menerima perintah lewat **teks dan suara**, bisa menjawab dengan **teks dan suara**, serta dapat membantu mengoperasikan komputer secara terbatas dan aman.

NARA menggunakan **OpenAI Codex** sebagai otak utama untuk pekerjaan coding, project analysis, command planning, dan bantuan developer. Namun fitur suara seperti wake word, speech-to-text, dan text-to-speech akan dibangun sebagai bagian dari sistem NARA sendiri.

---

## 1.3 One-liner

> NARA adalah asisten AI desktop untuk Windows 10 yang bisa dipanggil dengan suara, menjawab dengan suara, menampilkan HUD futuristik, dan membantu mengerjakan task coding maupun operasi komputer dengan sistem izin yang aman.

---

# 2. Visi Produk

Visi NARA adalah menjadi **AI desktop companion** yang terasa seperti asisten pribadi di komputer.

Pengguna bisa berkata:

```text
"NARA"
```

Lalu NARA aktif dan menjawab:

```text
"Ya, aku dengar."
```

Kemudian pengguna bisa memberi instruksi:

```text
"Buka project toko online-ku, cek error npm run dev, lalu jelaskan penyebabnya."
```

NARA akan:

```text
- Mendengar suara pengguna
- Mengubah suara menjadi teks
- Memahami instruksi
- Mengirim task coding ke Codex
- Meminta izin jika perlu menjalankan command
- Menampilkan proses di HUD
- Menjawab balik dengan suara
```

---

# 3. Masalah yang Ingin Diselesaikan

## 3.1 Masalah Utama

Banyak AI assistant sekarang masih berbentuk chat biasa. Pengguna tetap harus:

```text
- Mengetik prompt manual
- Copy-paste error
- Buka terminal sendiri
- Menjalankan command sendiri
- Pindah-pindah antara browser, VS Code, terminal, dan file explorer
- Membaca output error sendiri
```

NARA ingin membuat proses itu lebih natural:

```text
Pengguna bicara → NARA mendengar → NARA bekerja → NARA menjawab.
```

---

## 3.2 Masalah Developer Windows

Target awal NARA adalah pengguna Windows 10 yang ingin punya assistant AI lokal di desktop.

Masalah yang sering muncul:

```text
- Bingung cara menjalankan project
- Bingung membaca error terminal
- Sering lupa command
- Sulit memahami struktur repo
- Ingin AI membantu coding langsung di project
- Ingin tetap aman sebelum AI mengubah file atau menjalankan command
```

---

## 3.3 Masalah Voice Assistant Umum

Voice assistant biasa seperti Siri, Alexa, atau Google Assistant tidak fokus untuk coding dan project lokal.

NARA berbeda karena fokus pada:

```text
- Project coding
- Terminal command
- File system
- Git
- VS Code
- Windows app control
- HUD desktop
- Approval sebelum aksi berisiko
```

---

# 4. Tujuan Produk

## 4.1 Tujuan Utama

NARA bertujuan untuk:

```text
1. Menjadi AI assistant desktop untuk Windows 10.
2. Bisa menerima perintah lewat teks dan suara.
3. Bisa menjawab dengan teks dan suara.
4. Bisa dipanggil dengan nama "NARA".
5. Memiliki HUD desktop futuristik.
6. Menggunakan OpenAI Codex sebagai otak coding/agent.
7. Bisa membantu membaca project, menjalankan command, dan menjelaskan error.
8. Bisa membuka aplikasi Windows tertentu.
9. Memiliki approval system agar AI tidak bebas menjalankan aksi berbahaya.
10. Menyimpan log aktivitas agar semua aksi bisa diaudit.
```

---

## 4.2 Tujuan MVP

MVP pertama NARA harus bisa:

```text
- Berjalan di Windows 10.
- Memiliki HUD desktop.
- Bisa menerima input teks.
- Bisa menerima input suara minimal lewat tombol mic atau hotkey.
- Bisa menjawab dengan suara.
- Bisa terhubung ke OpenAI Codex.
- Bisa memilih folder project.
- Bisa mengirim task ke Codex.
- Bisa menampilkan output Codex di HUD.
- Bisa menjalankan command dengan approval.
- Bisa membaca git status dan git diff.
- Bisa menyimpan session dan log.
```

---

## 4.3 Target Ideal MVP

MVP ideal:

```text
User menekan hotkey atau tombol mic
        ↓
User bicara
        ↓
NARA transkrip suara menjadi teks
        ↓
NARA kirim instruksi ke Codex atau tool runtime
        ↓
NARA menampilkan proses di HUD
        ↓
Jika perlu command, NARA minta approval
        ↓
Setelah selesai, NARA menjawab dengan suara
```

---

# 5. Non-Goals

Untuk versi awal, NARA **tidak** akan:

```text
1. Membuat model AI sendiri.
2. Menjalankan model AI lokal seperti Ollama.
3. Mendukung Linux/macOS.
4. Menjadi full replacement Windows assistant.
5. Mengontrol mouse dan keyboard secara bebas.
6. Membaca password browser.
7. Mengambil credential Windows.
8. Mengedit registry Windows.
9. Menghapus file system penting.
10. Mengirim email/chat otomatis.
11. Melakukan transaksi finansial.
12. Mengoperasikan aplikasi sensitif tanpa izin.
13. Menjadi antivirus/security scanner.
14. Menjadi OS-level automation tanpa batas.
```

## 12.3 Kenapa Wake Word Bisa Jadi Stretch Goal?

Wake word yang selalu aktif lebih rumit karena harus:

```text
- Mendengarkan mic terus-menerus.
- Tidak boros CPU.
- Tidak sering salah aktif.
- Bisa membedakan "NARA" dari kata lain.
- Aman secara privasi.
- Tidak mengirim audio terus-menerus ke cloud.
```

Jadi urutan paling realistis:

```text
v0.1 = Push-to-talk voice
v0.2 = Hotkey voice
v0.3 = Wake word "NARA"
v0.4 = Always-listening stable mode
```


```text
OpenAI Speech-to-Text API
```

Daemon adalah backend lokal NARA.

Nama binary:

```text
nara-daemon.exe
```

Tugas:

```text
- Menjalankan local server.
- Mengatur session.
- Mengatur workspace.
- Menghubungkan HUD dengan Codex.
- Menjalankan tools.
- Mengelola approval.
- Mengelola voice pipeline.
- Menyimpan log.
```

---

Daemon harus bisa:

```text
- Start otomatis saat HUD dibuka.
- Berjalan di background.
- Menyediakan localhost API.
- Menyediakan WebSocket event stream.

- Menjalankan command Windows secara aman.
- Mengirim status real-time ke HUD.
- Menyimpan event log.
```
